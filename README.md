# ZANACMS Docs

This repository contains the documentation source for ZANACMS.

```text
help/   short help pages for the admin area and common user tasks
guide/  longer guides, reference documents and introductory texts
assets/ shared documentation assets
```

The published documentation site starts here:

- [Documentation start](https://gaxmann.github.io/zanacms-docs/)
- [Help overview](https://gaxmann.github.io/zanacms-docs/help/)
- [Guide overview](https://gaxmann.github.io/zanacms-docs/guide/)

The site uses the Jekyll theme `just-the-docs` with the custom dark colour scheme `zanadark`.
Custom documentation styling is kept in `_sass/custom/custom.scss`.
The navigation footer is configured in `_includes/nav_footer_custom.html`; the page footer is configured in `_includes/footer_custom.html` and reads the visible docs version from the root `version` file. The external CMS repository link is configured in `_config.yml`.
