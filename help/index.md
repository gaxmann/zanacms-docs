# ZANACMS Help

This is the short user help for ZANACMS. It is the right place to start when you are working in the admin area or setting up a website for the first time.

For the full technical documentation, start with the [Intro guide](../guide/index.md).

## Admin tools

- ☷ [Admin dashboard](adminbereich.md) — site status, current mode and installed version.
- ⚙ [Device management](zugangsverwaltung.md) — activate a browser or device for admin access.
- @ [Pages](seitenuebersicht.md) — edit pages in MD mode or Rich mode.
- ➔ [Navigation](navigationeditor.md) — edit the website menu.
- ⚃ [Media](medien.md) — upload images and copy ready-to-use image code.
- ⧉ [Backup/Update](backup.md) — create backups (run updates).

## Website setup

- [Configuration](konfiguration.md) — create and edit `/__config/conf.php`.
- [Modes](modi.md) — choose PHP mode, MD mode or Rich mode.
- [Links](links.md) — add internal and external links.
- [Footer](footer.md) — add footer links and translated footer text.
- [Images and media](bilder-und-medien.md) — use uploaded images in pages.
- [Page variables](seitenvariablen.md) — insert prepared values into MD or Rich pages.
- [Design and CSS](design-und-css.md) — adjust the site design safely.
