# ZANACMS Guide

This is the entry point for the longer ZANACMS documentation. Start with the installation page if you are setting up a site for the first time.

If you know nothing about ZANACMS yet, start with the [README](https://github.com/gaxmann/zanacms/blob/master/README.md) of the main CMS repository. It contains the general project overview, repository context and basic information before the detailed guide starts.

- [Introduction]({{ '/guide/intro/' | relative_url }}) — structure, modes and shared concepts.
- [PHP mode]({{ '/guide/php-mode/' | relative_url }}) — developer-controlled PHP pages.
- [Markdown mode]({{ '/guide/markdown-mode/' | relative_url }}) — Markdown files and the Markdown editor.
- [Rich mode]({{ '/guide/rich-mode/' | relative_url }}) — page content maintained through the rich text editor.
